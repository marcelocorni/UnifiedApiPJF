# Unified API

## Descrição

Este projeto implementa uma API unificada para centralizar e simplificar integrações de múltiplos serviços e dados.

## Objetivo

Fornecer uma interface consistente e eficiente para consumo de dados heterogêneos, reduzindo complexidade de integrações e padronizando requisições.

## Stack Tecnológico

- Linguagem: C#
- Framework: .Net 8.0
- Banco de Dados: PostgreSQL (Neon)
